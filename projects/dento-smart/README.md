# Dento Smart — Smart Dental Equipment Monitoring System

> **Dento Smart** is an IoT-based smart monitoring system for dental equipment, developed by PT Media Teknologi Celebes in 2026. It monitors operational parameters of dental chair and equipment systems, providing real-time visibility via a web dashboard and backend API.
