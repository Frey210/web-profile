# Fire Fighting Robot — Tech Stack

| Component | Technology |
|---|---|
| MCU | STM32 (ARM Cortex-M) |
| Language | C (STM32 HAL / bare metal) |
| IDE | STM32CubeIDE |
| Servo Control | PWM (12 channels) |
| Sensors | UV/IR Flame sensor, Ultrasonic |
| Locomotion | Hexapod tripod gait algorithm |
| Suppression | DC Fan / Water Pump |
