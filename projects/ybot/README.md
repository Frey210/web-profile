# Y-BOT — SLAM Mobile Robot

> **Y-BOT** is an autonomous mobile robot built using ROS (Robot Operating System) and the SLAM (Simultaneous Localization and Mapping) algorithm with a Microsoft Kinect depth sensor. It serves as the subject of Fariz's undergraduate thesis and a published journal paper.
