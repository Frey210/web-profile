# Ergoquipt — Technology Stack

## Embedded / Firmware
| Component | Technology |
|---|---|
| Microcontroller | ESP32, Arduino |
| Language | C++ (Arduino Framework) |
| Temperature Sensor | MLX90614 (infrared tympanic) |
| Heart Rate Sensor | PPG sensor module |
| Reaction Time | Physical button + microsecond hardware timer |
| Communication | Wi-Fi (HTTP/MQTT to backend) |

## Backend
| Component | Technology |
|---|---|
| API Framework | Node.js (Express) + FastAPI (Python) |
| Database | PostgreSQL |
| Auth | JWT |
| API Documentation | Swagger / OpenAPI |

## Frontend / Web
| Component | Technology |
|---|---|
| Framework | React.js |
| Real-time | WebSocket or polling |

## Mobile
| Component | Technology |
|---|---|
| Framework | React Native / Flutter |
| Platforms | Android, iOS |

## Infrastructure
| Component | Technology |
|---|---|
| OS | Ubuntu / Debian Linux |
| Containers | Docker, Docker Compose |
| Proxy | Nginx |
| Access | Cloudflare Tunnel |
