# Ergoquipt — Ergoquipt Reaction Time & Physiological Measurement System

> **Ergoquipt** is a multi-device reaction time and physiological monitoring platform developed by PT Media Teknologi Celebes for ergonomics research. It measures tympanic (ear) temperature, heart rate, and reaction times across multiple test subjects simultaneously, using embedded devices and a real-time web interface.

## Key Capabilities

- Millisecond-precise reaction time measurement
- Tympanic (infrared ear) temperature measurement
- Heart rate monitoring
- Multi-device synchronized testing
- Real-time web admin dashboard
- REST API backend with OpenAPI docs
- Mobile application for test management
