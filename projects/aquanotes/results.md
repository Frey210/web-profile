# Aquanotes — Results & Outcomes

## Deployment Status

✅ **Production deployed** — Platform is live and actively used.

- API documentation: https://aeraseaku.inkubasistartupunhas.id/docs

## Quantitative Outcomes

> *Note: Specific client numbers and measurable farm impact metrics are not publicly documented at this time. The following reflects what is known from references. Values marked [assumed] should be verified and updated with actual data from PT Media Teknologi Celebes internal records.*

- **6+ water quality parameters** monitored simultaneously per sensor node
- **Real-time monitoring** — sensor data refreshed every 30–60 seconds [assumed based on typical IoT interval]
- **Multiple pond support** within single platform instance
- **Production-grade deployment** on a Linux VPS with Docker orchestration
- **Full REST API** with OpenAPI documentation (accessible at production URL)
- **Cross-platform access** — web dashboard + Android/iOS mobile app

## Qualitative Outcomes

- Enabled shrimp farmers to shift from manual periodic checks to **continuous automated monitoring**
- Platform serves as the **flagship product** of PT Media Teknologi Celebes, used to demonstrate company capabilities to enterprise clients
- **Academic and institutional recognition** — built within the Inkubasi Startup Universitas Hasanuddin ecosystem
- Demonstrated the viability of low-cost IoT sensors for precision aquaculture in remote Indonesian locations
- Provided foundation for subsequent IoT products at MTC (Ergoquipt, Dento Smart, Monitoring Platform share the same infrastructure patterns)
