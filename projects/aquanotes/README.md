# Aquanotes — IoT Aquaculture Monitoring Platform

> **Aquanotes** is a full-stack IoT platform developed by PT Media Teknologi Celebes (Aerasea) for real-time monitoring of shrimp pond water quality. It connects embedded sensor nodes in aquaculture ponds to a cloud backend, web dashboard, and mobile application.

## Monitored Parameters

- 🌡️ Water Temperature
- 💧 TDS (Total Dissolved Solids / Salinity)
- 🫧 Dissolved Oxygen (DO)
- ⚗️ pH Level
- ☣️ Ammonia Concentration
- 🌤️ Environmental Parameters
