# Overview

## Problem Statement

Citizens in Indonesian municipalities face challenges in reporting issues (potholes, broken infrastructure, waste management, public safety concerns) to local government. Traditional channels are slow, opaque, and lack feedback mechanisms. A digital citizen reporting platform can streamline this process and improve government responsiveness.

## Solution

SiWarga ("Si Warga" = "The Citizen" in Indonesian) is a web-based platform enabling citizens to report community issues, track the status of their reports, and receive feedback from responsible government units.

## Key Features

- Citizen issue reporting with location data and photo attachments [assumed based on standard civic reporting platforms]
- Report status tracking (submitted → in review → resolved)
- Category-based issue classification (infrastructure, sanitation, safety, etc.)
- Government admin dashboard for report management
- Web-based, responsive interface
