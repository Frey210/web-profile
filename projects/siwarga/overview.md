# SiWarga — Project Overview

## Problem Statement

Indonesian citizens lack accessible, transparent channels to report community issues to local government. Traditional reporting (phone hotlines, in-person visits) is slow, undocumented, and provides no status feedback. This results in unreported issues remaining unresolved and citizens feeling disconnected from local government responsiveness.

## Solution

SiWarga is a civic-tech web application enabling citizens to digitally report community issues (infrastructure damage, sanitation problems, public safety concerns) with their location, photos, and descriptions. Reports are routed to responsible government units, who can update status and communicate with reporters.

## Target Users

- Citizens of a municipality (kelurahan/kecamatan level)
- Local government administrators (admin panel for managing reports)

## Key Features

- Digital issue reporting with location and category
- Photo attachment support
- Report status tracking for citizens
- Admin dashboard for government response management
- Mobile-responsive web interface
