# Airport Project — Tech Stack

## Electronic Systems

| System | Technology / Brand |
|---|---|
| Data Network | Cisco (Catalyst switches, access points, IOS configuration) |
| Baggage Handling PLC | Allen-Bradley (Rockwell Automation) |
| BHS SCADA/HMI | Rockwell FactoryTalk (or equivalent) |
| Fire Alarm System | Notifier (Honeywell) — Addressable FAS |
| Public Address | TOA, Bosch — Zoned PA amplification |
| X-Ray Hold Baggage Screening | Leidos HBS |
| Structured Cabling | CAT6A UTP, Single-mode/multi-mode Fiber Optic |

## Engineering Standards

| Standard | Application |
|---|---|
| ICAO Standards & Recommended Practices | Overall aviation safety compliance |
| IEC 60839 | Electronic security and alarm systems |
| TIA-568 | Structured cabling standards |
| NFPA 72 | Fire Alarm and Signaling Code (reference) |
| SNI (Standar Nasional Indonesia) | National standards for airport construction |

## Tools Used

| Tool | Purpose |
|---|---|
| Cisco IOS CLI | Switch/router configuration and troubleshooting |
| Fluke cable certifier | Structured cabling test and certification |
| RSLogix / Studio 5000 | Allen-Bradley PLC program review |
| Non-Conformance Report (NCR) system | Quality deficiency tracking |
| Microsoft Excel / Word | Test record and inspection documentation |
