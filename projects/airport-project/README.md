# Overview

## Mission

To ensure that all electronic systems installed in the Sultan Hasanuddin International Airport expansion meet international aviation standards, quality requirements, and contractual specifications through systematic supervision, testing, and commissioning oversight.

## Problem Context

Airport expansion projects involve dozens of electronic subsystems from multiple vendors, installed by multiple contractors. Without rigorous quality assurance oversight, systems may be installed incorrectly, without proper testing, or without meeting aviation regulatory standards — creating safety risks for passengers, aircraft, and airport operations.

## Scope of Responsibility

Fariz's QA role covered the following electronic system categories:

### 1. Data Network Infrastructure
- Cisco-based structured cabling (CAT6/fiber) and active network equipment
- Network switch configuration and troubleshooting
- Airport-wide LAN/WLAN design validation

### 2. PLC & SCADA — Baggage Handling System (BHS)
- Allen-Bradley PLC programming validation for conveyor and sorting systems
- SCADA HMI configuration and testing
- BHS functional acceptance testing and commissioning

### 3. Fire Alarm System (FAS)
- Notifier brand fire detection and alarm system
- Full terminal/panel loop integrity testing
- Zone coverage and evacuation function validation

### 4. Public Address System (PAS)
- TOA and Bosch PA system integration
- Zone amplifier configuration
- Announcement quality and zone routing testing

### 5. X-Ray Hold Baggage Screening (HBS)
- Leidos HBS X-ray system commissioning and acceptance
- Image quality verification, threat detection calibration support

# Contribution

## Role
**Electronic Officer — Quality Assurance (Owner's Representative)**

Fariz represents PT Angkasa Pura I's quality interests against the contractor's work. This is an **owner's QA role** — Fariz is not installing systems himself but is responsible for verifying they are installed, tested, and commissioned correctly.

## Key Contributions

- Supervised and witnessed commissioning of PLC/SCADA BHS systems — reviewed ladder logic programs, witnessed functional tests, signed off on test records
- Performed and witnessed structured cabling tests on airport data network (cable certification, patch panel verification)
- Configured and troubleshot Cisco network equipment (switch configuration, VLAN setup, connectivity verification)
- Reviewed and approved fire alarm zone layouts, panel loop tests, and zone-by-zone functional tests (Notifier FAS)
- Supported PA system zone configuration (TOA/Bosch) and announcement routing verification
- Witnessed X-Ray HBS (Leidos) commissioning and image quality calibration
- Prepared quality documentation: inspection records, non-conformance reports (NCR), acceptance certificates

## Value Delivered

This role provides experience that is extremely rare for a junior engineer of Fariz's age:
- Worked on safety-critical systems in a regulated aviation environment
- Gained exposure to enterprise-grade electronic systems from global brands (Cisco, Leidos, Notifier, Rockwell, Bosch)
- Developed QA/QC methodology for complex multi-vendor system integration
