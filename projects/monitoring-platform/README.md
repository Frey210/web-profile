## Key Capabilities

- Real-time server resource monitoring (CPU, RAM, Disk, Network)
- IoT device connectivity and health tracking
- Domain and SSL certificate expiry monitoring
- Container status monitoring (Docker)
- Log aggregation and error alerting
- Public-facing status dashboard

## Technology Stack

`Grafana` `Prometheus` `Docker` `Node Exporter` `Nginx` `Cloudflare Tunnel` `Linux (Ubuntu/Debian)`
