# RadioLink IoT — Overview

## Problem Statement

Aviation landing system instruments in remote field locations need temperature monitoring without wired connectivity back to control centers.

## Solution

IoT temperature monitoring system using radio link (LoRa/RF) for wireless sensor-to-dashboard communication. Remote sensor nodes at instrument cabinets transmit temperature readings wirelessly to a central receiver for real-time monitoring.
